\LANCER LE BINGO SONORE/

!! IMPORTANT !!

Avant de faire quoi-que ce soit, il faut impérativement que Node.JS soit installé sur l'ordinateur.
Lien de téléchagement : https://nodejs.org/dist/v10.13.0/node-v10.13.0-x64.msi
Ainsi que Git.
Lien de téléchagement : https://git-scm.com/download/win


Dans le dossier Code, vous trouverez un fichier nommé "!Démarrer le Bingo".
Il vous suffit de double cliquer dessus.
Une page Internet sera automatiquement ouverte, une autre fenêtre avec un fond noir, c'est cette fenêtre
fera fonctionner le Bingo.

Une fois la fenêtre Internet ouverte, le site devrait d'abord afficher "Ce site est inaccessible", il faut
attendre le temps que tout se lance, raifraichissait la page de temps en temps jusqu'a que le site apparaisse.

Une fois le site lancé, il y aura  un bouton "Start", il suffit de cliquer dessus pour lancer le Bingo.
Il faudra re-cliquer dessus à chaque fois pour relancer un son.
